# 🧠 UniBot: PDF QA Chat

UniBot is an AI-powered chatbot that allows users to upload PDF documents and ask questions about their contents. Built using LangChain, OpenAI, Pinecone, and React, UniBot enables seamless document comprehension and interactive Q&A.

---

## 🚀 Features

- 📄 Upload and ingest PDFs using LangChain
- 💬 Ask questions about the document via a user-friendly chat UI
- 🧠 OpenAI embeddings with Pinecone vector store for semantic retrieval
- ⚡ FastAPI backend with Vite + React frontend
- 🔍 Real-time response generation based on your queries
